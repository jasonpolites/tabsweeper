Customize Behavior

Specify a URL pattern to match specific websites, then choose what happens when Tab Sweeper is clicked ?
  • URL Pattern
    A string (text) pattern to identify a tab and assign a desired behavior

    Examples:
    https://www.reddit.com Match any url containing https://www.reddit.com
    google.com Match any url containing google.com
    (e.g. will include www.google.com and drive.google.com)
    /^https:\/\/drive\.google\.com.*/ Use / at the start and end to denote a "regular expression". This example will match URLs starting with https://drive.google.com, but will not match URLs starting with https://www.google.com
  • Use Full URL
    If checked, the full URL (including query params) will be considered when matching.
    Default is false (unchecked)
  • Detect Duplicates Using
    The match logic to use when determining duplicates.
    By default, both the URL and page title together will be used.
