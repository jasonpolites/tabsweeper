Exclusions

You can customize the duplicate matching based on a URL pattern (Help)
  • URL Pattern
    A regular expression used to determine which URLs to match. (eg http://www.google.com or google\.*)
  • Use Full URL
    If checked, the full URL (including query params) will be considered when matching. Default is false (unchecked)
  • Behavior
    The match logic to use when determining duplicates
